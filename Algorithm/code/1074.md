### 풀이 1
재귀함수를 통해 주어진 행렬을 모두 방문하는 코드를 작성했다.
시간 + 메모리 초과
```
def make_map(arr, n):
  maps = []
  base = [(0,0), (0,n), (n,0), (n,n)]
  for b in base:
    for a in arr:
      maps.append((b[0] + a[0], b[1] + a[1]))
  return maps

N ,r, c = map(int, input().split())

n = 1
arr = [(0, 0), (0, 1), (1, 0), (1, 1)]
while len(arr) < 2 ** (2 * N):
  n *= 2
  arr = make_map(arr, n)
print(arr.index((r, c)))
```
